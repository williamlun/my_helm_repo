# Workflow
1. Create a new chart
```
helm create mychart
```
Edit the chart and values files for your images.

2. Package the chart
```
helm package mychart
```
3. Update the index.yaml file
```
helm repo index --url https://williamlun.github.io/my_helm_repo .
```
4. Push the chart and index.yaml to the chart repository
```
git add .
git commit -m "Add mychart"
git push
```

5. Update the Helm client cache
```
helm repo update
```
